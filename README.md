# Smart Sports System - Graduation Project

An advanced, AI-powered sports management and analytics platform. This system integrates real-time video analysis, custom nutrition planning, AI chatbots, and club management tools to elevate athletic performance and team administration.

##  Features

- **AI Video Analysis:** Real-time feedback on exercise form and technique using Computer Vision.
- **Smart Nutrition:** AI-generated meal plans tailored to athlete stats.
- **Intelligent Chatbot:** An LLM-powered RAG chatbot for medical and sports guidance.
- **Club & CV Management:** Dashboards for clubs to scout players, manage job offers, and analyze player CVs.

##  Tech Stack

- **Frontend:** React.js, Axios
- **Backend:** FastAPI, PostgreSQL, SQLAlchemy, Alembic
- **AI & Machine Learning:** PyTorch, LangGraph, Computer Vision
